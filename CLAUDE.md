# Soren Grammar

## Project Overview
An interactive grammar game for Soren.

## Git & GitHub Workflow

**Commit and push regularly as work progresses.** After any meaningful change — new feature, bug fix, content addition, or structural update — commit with a clean message and push to GitHub. Never let significant work sit uncommitted.

### Commit message format
```
<type>: <short description>

Examples:
feat: add drag-and-drop sentence builder
fix: correct answer validation for plurals
content: add 10 new noun exercises
style: improve button layout on mobile
```

### Workflow
```bash
git add <files>
git commit -m "type: description"
git push origin main
```

Push early and often so work is never lost and progress is always visible on GitHub.

## Tech Stack
- HTML/CSS/JavaScript (single-file or multi-file as needed)
- Hosted via GitHub Pages

## Development Notes
- Keep the game playable and accessible at the GitHub Pages URL at all times
- Test in browser before committing UI changes
